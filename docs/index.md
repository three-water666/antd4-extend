# Antd4 Extend

`antd4-extend` 是一个专门为 Ant Design v4 用户设计的组件扩展库。

### 为什么需要它？

随着 Ant Design v5 的发布，许多优秀的新组件（如 `Tour` 漫游导览、`FloatButton` 悬浮按钮等）被引入。然而，由于 v5 采用了全新的 CSS-in-JS 样式方案，许多仍在使用 v4 版本的项目面临着巨大的迁移压力。

`antd4-extend` 的目标是：
- **向后兼容**：在 v4 环境中直接使用 v5+ 的组件。
- **一致性**：保持与 v4 相同的 Less 变量支持和样式风格。
- **轻量级**：只引入你需要的组件，不增加额外负担。
